Arbetsprocessen

Jag använder Visual Studio Code (VSC) för utveckling och kör även npm-delarna i terminalen på VSC.

Utveckling

  • Skapar mapp lokalt på datorn för projektet
  • Använder terminalen i VSC för att installera parcel --save dev
  • Ställer in src-mapp för index.html i package-json filen
  • Skapar en .gitignore för node-modules, dist och parcel-cache
  • Skapar ett GitRepo
  • Skapar en sida på Render
  • Kopplar ihop Render med GitHub för kommande automatisk uppdatering/publicering
  • Skapar en developerbranch i Git-repot
  • Lägger till Browserslist i package.json-filen för bakåtkompabilitet
  • Kodar HTML, CSS och JavaScript
  • Lägger till mapp med bilder
  • Kör npm run build i terminalen vilket optimerar bildfilerna och transpilerar och minifierar övriga html, css och js-filer

Publicering

  • Mergar developerbranch med mainbranch i Gitrepot
  • GitHub och Render synkas och publicerar webbplatsen
